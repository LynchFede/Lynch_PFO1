# PFO 1 - Portafolio Personal

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un **Portafolio Personal** utilizando **HTML5 y CSS3**. El objetivo es aplicar los conceptos fundamentales de maquetación web, estructura semántica, formularios, imágenes, enlaces y estilos CSS para construir una página web organizada, accesible y responsive.

La página incluye información personal, proyectos destacados, habilidades, un formulario de contacto y una sección de películas favoritas.

## Uso de Google Fonts

Se incorporó una fuente externa mediante Google Fonts para mejorar la presentación visual y la legibilidad del sitio.

### ¿Por qué elegiste esta fuente?

Elegí la fuente **Poppins** porque posee un diseño moderno, limpio y de fácil lectura. Consideré que era adecuada para un portafolio personal porque mejora la presentación visual del contenido y brinda una apariencia profesional al sitio web.

---

## Organización del Layout

Se utilizó **Flexbox** para organizar las tarjetas de proyectos y distintos bloques de contenido de forma ordenada y adaptable.

### ¿Qué ventajas encontraste al utilizar Flexbox o Grid en tu proyecto?

Utilicé Flexbox para organizar las tarjetas de proyectos y otros elementos de la página. Su principal ventaja fue la facilidad para distribuir el contenido de manera ordenada y adaptable a diferentes tamaños de pantalla. Además, permitió alinear elementos horizontal y verticalmente con pocas líneas de código, simplificando el diseño responsivo del sitio.

---

## Animaciones y Transiciones

Se implementaron efectos visuales utilizando transiciones CSS para mejorar la interacción con el usuario.

### ¿Qué animación o transición implementaste y por qué consideraste que era adecuada para tu proyecto?

Implementé una transición de efecto **hover** en botones y enlaces. Cuando el usuario posiciona el cursor sobre ellos, cambian suavemente de color, mejorando la interacción visual y proporcionando una experiencia de navegación más agradable. Elegí esta transición porque aporta dinamismo al sitio sin afectar la legibilidad ni distraer al usuario del contenido principal.
